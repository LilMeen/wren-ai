# Database Description

| Tên bảng | Nguồn | Mô tả | Relationship |
|---|---|---|---|
| `raw_dmp_evt_connectivity` |  | Dữ liệu raw sự kiện connectivity/heartbeat của thiết bị: trạng thái online/offline, quality score, lý do offline, thời điểm ghi nhận. | Xem chi tiết thiết bị tại `dim_device` theo `deviceId` = `device_id`. |
| `raw_dmp_tlm_raw` |  | Dữ liệu raw telemetry theo thiết bị, chứa payload `telemetry` dạng JSON và thông tin loại thiết bị/thời gian sự kiện. | Xem chi tiết thiết bị tại `dim_device` theo `deviceId` = `device_id`. |
| `raw_dmp_public_asset` |  | Dữ liệu raw asset từ DMP: asset id, tên, label, type, tenant/customer, profile, external id. | Xem profile asset tại `raw_dmp_public_asset_profile` theo `asset_profile_id` = `id`. |
| `raw_dmp_public_asset_profile` |  | Dữ liệu raw asset profile từ DMP: cấu hình/profile mặc định, dashboard/rule chain/queue liên quan. |  |
| `raw_dmp_public_device` |  | Dữ liệu raw device từ DMP: device id, tên, label, type, tenant/customer, firmware/software, profile, device data. | Xem profile thiết bị tại `raw_dmp_public_device_profile` theo `device_profile_id` = `id`. |
| `raw_dmp_public_device_profile` |  | Dữ liệu raw device profile từ DMP: loại profile, transport/provision type, profile data, rule chain/dashboard/queue. |  |
| `raw_dmp_public_relation` |  | Dữ liệu raw quan hệ giữa entity trong DMP, gồm `from_id`, `to_id`, loại entity và loại quan hệ. | Mapping entity nguồn/đích theo `from_id`, `from_type`, `to_id`, `to_type`; nếu là DEVICE/ASSET thì đối chiếu với raw device/asset theo `id`. |
| `raw_parking_db_vehicle_histories` |  | Dữ liệu raw lịch sử xe vào/ra bãi: biển số, bãi đỗ, làn/cổng, thời gian check-in/out, phí, thanh toán, trạng thái. | Xem bãi đỗ tại `dim_parking_lot` theo `pk_lot_id`; mapping event sang `fct_vehicle_events` theo `id` = `event_id`. |
| `stg_dmp_asset_profiles` | `raw_dmp_public_asset_profile` | Bảng staging chuẩn hóa asset profile: đổi tên `id` thành `asset_profile_id`, chuẩn hóa timestamp/load metadata. | Mapping về raw theo `asset_profile_id` = `raw_dmp_public_asset_profile.id`; xem asset dùng profile tại `stg_dmp_assets` theo `asset_profile_id`. |
| `stg_dmp_assets` | `raw_dmp_public_asset` | Bảng staging chuẩn hóa asset: asset id, tên/label/type, tenant/customer, profile, metadata load. | Xem profile tại `stg_dmp_asset_profiles` theo `asset_profile_id`; xem dim tại `dim_asset` theo `asset_id`. |
| `stg_dmp_device_profiles` | `raw_dmp_public_device_profile` | Bảng staging chuẩn hóa device profile: cấu hình loại thiết bị, transport/provision, firmware/software, rule chain/dashboard. | Mapping về raw theo `device_profile_id` = `raw_dmp_public_device_profile.id`; xem device dùng profile tại `stg_dmp_devices` theo `device_profile_id`. |
| `stg_dmp_device_status_events` | `raw_dmp_evt_connectivity` | Bảng staging sự kiện trạng thái thiết bị, chuẩn hóa current/previous status, event time/date/hour, flag online/status-change. | Xem chi tiết thiết bị tại `dim_device` theo `device_id`; có thể đối chiếu connectivity raw theo `device_id` = `deviceId`. |
| `stg_dmp_devices` | `raw_dmp_public_device` | Bảng staging chuẩn hóa device: device id, tên/label/type, tenant/customer, profile, firmware/software, device data. | Xem profile tại `stg_dmp_device_profiles` theo `device_profile_id`; xem dim tại `dim_device` theo `device_id`. |
| `stg_dmp_evt_connectivity` | `raw_dmp_evt_connectivity` | Bảng staging connectivity event từ DMP: device, tenant/customer, trạng thái, lý do offline, quality score, timestamp. | Xem chi tiết thiết bị tại `dim_device` theo `deviceid` = `device_id`. |
| `stg_dmp_relations` | `raw_dmp_public_relation` | Bảng staging quan hệ entity DMP, giữ `from_id`, `to_id`, entity type và relation type. | Dùng để tạo quan hệ device-asset; đối chiếu `from_id`/`to_id` với `stg_dmp_devices.device_id` hoặc `stg_dmp_assets.asset_id` theo `from_type`/`to_type`. |
| `stg_vehicle_histories` | `raw_parking_db_vehicle_histories` | Bảng staging lịch sử xe, chuẩn hóa event id, biển số, bãi/cổng/làn, thời gian vào/ra, phí, thanh toán, trạng thái. | Xem bãi đỗ tại `dim_parking_lot` theo `pk_lot_id`; xem fact chuẩn hóa tại `fct_vehicle_events` theo `event_id`. |
| `dim_asset` | `stg_dmp_assets`, `stg_dmp_asset_profiles` | Dimension asset đã enrich profile: thông tin asset, loại/profile, tenant/customer, rule chain/dashboard/queue. | Xem asset-device mapping tại `dim_device_asset` theo `asset_sk` hoặc `asset_id`; xem assignment fact tại `fct_device_asset_assignment` theo `asset_sk` hoặc `asset_id`. |
| `dim_asset_profile` | `stg_dmp_asset_profiles` | Dimension asset profile: mô tả/cấu hình profile asset, default flag, rule chain/dashboard/queue. | Xem asset dùng profile tại `dim_asset` theo `asset_profile_id` nếu cột này được expose ở dim/source. |
| `dim_date` |  | Dimension ngày dùng cho phân tích theo date key, ngày/tháng/quý/năm, week, weekend/weekday. | Mapping với `fct_vehicle_events.check_in_date_key`, `fct_vehicle_events.check_out_date_key`, `fct_parking_occupancy.occupancy_date_key` theo `date_key`. |
| `dim_device` | `stg_dmp_devices`, `stg_dmp_device_profiles` | Dimension thiết bị đã enrich profile: device id/name/type, tenant/customer, firmware/software, profile, transport/provision. | Xem profile tại `dim_device_profile` theo `device_profile_id`; xem asset gắn với thiết bị tại `dim_device_asset` theo `device_sk` hoặc `device_id`; xem assignment tại `fct_device_asset_assignment` theo `device_sk` hoặc `device_id`. |
| `dim_device_asset` | `dim_device`, `dim_asset`, `stg_dmp_relations` | Dimension bridge hiện tại giữa thiết bị và asset, đã flatten thông tin device + asset + profile. | Xem chi tiết thiết bị tại `dim_device` theo `device_sk` hoặc `device_id`; xem chi tiết asset tại `dim_asset` theo `asset_sk` hoặc `asset_id`. |
| `dim_device_asset_snapshot` | `dim_device_asset` | Snapshot/SCD lịch sử quan hệ device-asset, có `dbt_valid_from`, `dbt_valid_to`, `dbt_scd_id`. | Xem bản hiện tại tại `dim_device_asset` theo `device_sk` + `asset_sk`; xem thiết bị tại `dim_device` theo `device_sk`; xem asset tại `dim_asset` theo `asset_sk`. |
| `dim_device_profile` | `stg_dmp_device_profiles` | Dimension device profile: loại profile, transport/provision, firmware/software, profile data, rule chain/dashboard/queue. | Xem thiết bị dùng profile tại `dim_device` theo `device_profile_id`. |
| `dim_parking_lot` | `stg_vehicle_histories` | Dimension bãi đỗ xe, gồm mã bãi, tên bãi và khu vực. | Xem event xe tại `fct_vehicle_events` theo `pk_lot_id` = `parking_lot_id`; xem occupancy tại `fct_parking_occupancy` theo `pk_lot_id` = `parking_lot_id`. |
| `dim_parking_lot_snapshot` | `dim_parking_lot` | Snapshot/SCD lịch sử bãi đỗ, có thời gian hiệu lực của tên bãi/khu vực. | Xem bãi hiện tại tại `dim_parking_lot` theo `pk_lot_id`. |
| `dim_time` |  | Dimension thời gian trong ngày: hour, minute, label, period, minutes since midnight. | Mapping với `fct_vehicle_events.check_in_time_key`, `fct_vehicle_events.check_out_time_key`, `fct_parking_occupancy.occupancy_time_key` theo `time_key`. |
| `fct_device_asset_assignment` | `dim_device`, `dim_asset`, `stg_dmp_relations` | Fact quan hệ/gán thiết bị vào asset, gồm device/asset key/id, tenant, relation type và version. | Xem chi tiết thiết bị tại `dim_device` theo `device_sk` hoặc `device_id`; xem chi tiết asset tại `dim_asset` theo `asset_sk` hoặc `asset_id`. |
| `fct_vehicle_events` | `stg_vehicle_histories`, `dim_date`, `dim_time`, `dim_parking_lot` | Fact chi tiết từng lượt xe/check-in/check-out: thời gian, bãi, loại xe, payment, phí, duration, cổng/làn, trạng thái. | Xem bãi đỗ tại `dim_parking_lot` theo `parking_lot_id` = `pk_lot_id`; xem ngày tại `dim_date` theo `check_in_date_key`/`check_out_date_key` = `date_key`; xem giờ tại `dim_time` theo `check_in_time_key`/`check_out_time_key` = `time_key`. |
| `fct_parking_occupancy` | `fct_vehicle_events`, `dim_date`, `dim_time`, `dim_parking_lot` | Mart aggregate occupancy theo bãi, loại xe, ngày/giờ: số xe vào, số xe ra, current occupancy. | Xem bãi đỗ tại `dim_parking_lot` theo `parking_lot_id` = `pk_lot_id`; xem ngày tại `dim_date` theo `occupancy_date_key` = `date_key`; xem giờ tại `dim_time` theo `occupancy_time_key` = `time_key`. |
| `stg_mv_dmp_tlm_camera` | `raw_dmp_tlm_raw` | Materialized view telemetry camera, parse JSON telemetry thành CPU/memory/fan/heater/reboot/uptime theo thiết bị. | Xem chi tiết thiết bị tại `dim_device` theo `deviceId` = `device_id`. |
| `stg_mv_dmp_tlm_chiller` | `raw_dmp_tlm_raw` | Materialized view telemetry chiller, parse trạng thái chiller/fault/mode/valve limit theo thiết bị. | Xem chi tiết thiết bị tại `dim_device` theo `deviceId` = `device_id`. |
| `stg_mv_dmp_tlm_energy_meter` | `raw_dmp_tlm_raw` | Materialized view telemetry energy meter, parse dòng điện, điện năng, tần số, công suất, hệ số công suất, điện áp, nước. | Xem chi tiết thiết bị tại `dim_device` theo `deviceId` = `device_id`. |
| `stg_mv_dmp_tlm_nvr` | `raw_dmp_tlm_raw` | Materialized view telemetry NVR, parse CPU/memory/uptime theo thiết bị. | Xem chi tiết thiết bị tại `dim_device` theo `deviceId` = `device_id`. |
