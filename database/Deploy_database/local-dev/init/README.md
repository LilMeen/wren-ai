# init/

Files in this folder are mounted into `starrocks-data-loader` at `/docker-entrypoint-initdb.d/`
and executed in alphabetical order on first `docker compose up`.

| File | Description |
|------|-------------|
| `01_schema.sql` | Creates database `sdp_near_realtime` and both tables |
| `02_mock_data.sql` | 1-month hourly mock data (generated by `gen_mock.py`) |

## Regenerate mock data

```bash
cd database/mock_data
python gen_mock.py
copy mock_data.sql ..\Deploy_database\local-dev\init\02_mock_data.sql
```

Or use the helper script at the root of `local-dev/`:

```powershell
.\prepare-init.ps1
```
