# init/

Files in this folder are mounted into `starrocks-data-loader` at `/docker-entrypoint-initdb.d/`
and executed in alphabetical order on first `docker compose up`.

| File | Description |
|------|-------------|
| `01_schema.sql` | DDL for all 17 tables (Hive raw + Iceberg staging/golden) |
| `02_mock_data.sql` | Mock data for all 17 tables (generated by `gen_mock.py`) |

## Regenerate schema + mock data

```bash
cd database/mock_data
python gen_mock.py
```

`gen_mock.py` writes both `schema.sql` → `init/01_schema.sql` and `mock_data.sql` → `init/02_mock_data.sql` automatically.

Or use the helper script at the root of `local-dev/`:

```powershell
.\prepare-init.ps1
```
